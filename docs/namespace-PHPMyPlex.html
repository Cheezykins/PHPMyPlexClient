Classes summary
MyPlex

The MyPlex class is used to provide connectivity to the MyPlex API. It handles the login and authentication tokens needed for subsequent calls. It can also provide a list of servers available within the account.
PlexServer

Models the plex server. Provides the basis of requesting sections and subsections within each server as well as high level server details.
PlexServerCollection Provides an itterable array of servers you can call by name.
Proxy Defines a proxy for sending requests to plex.
Request

Provides an interface to the Httpful request library to communicate with plex. Defines custom headers that Plex requires as part of its API specification.
